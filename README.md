# SmartTrade Vision (Frontend)

This is a minimal Vite + React + Tailwind project prepared for you.
Logo is embedded at `public/assets/logo.png`.

How to use:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run dev server:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

Deploy to Vercel: Import this repository into Vercel (it will auto-detect Vite/React). Use `npm run build` as the build command and `build` as output directory.
